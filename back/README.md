## Running the app*

```bash
# development with nodemon
$ npm i nodemon --save-dev
$ nodemon --watch "src/**" --ext "ts,json" --ignore "src/**/*.spec.ts" --exec "ts-node src/main.ts"

# development
$ npm run start  or nest start

# watch mode
$ npm run start:dev  

# production mode
$ npm run start:prod
```
